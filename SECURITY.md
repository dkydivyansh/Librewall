# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| <1.0  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of **Librewall** seriously. If you have discovered a security vulnerability, please report it immediately so we can address it.

### How to Submit a Report

1. Go to the **[Issues](../../issues)** tab of this repository.
2. Click on the green **New Issue** button.
3. Fill out the issue description with the details below.
4. **Add the Label:** On the right-hand sidebar, click on **Labels** and select **`Vulnerability`**.
   * *If you do not have permission to add labels, please prefix your issue title with `[VULNERABILITY]`.*

### What to Include

To help us resolve the issue quickly, please include:
* **Type of Vulnerability** (e.g., XSS, RCE, Path Traversal)
* **Severity Level** (Low, Medium, High, Critical)
* **Description:** A detailed explanation of the issue.
* **Steps to Reproduce:** Step-by-step instructions to replicate the vulnerability.
* **Proof of Concept (PoC):** Screenshots or code snippets (if applicable).
